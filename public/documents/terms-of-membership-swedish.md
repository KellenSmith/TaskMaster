# Medlemsvillkor

**Senast uppdaterad:** 20 februari 2026

Dessa medlemsvillkor gäller för alla användare som väljer att bli medlemmar i TaskMaster (demoversion).

## 1. Avtal

Genom att bli medlem i TaskMaster samtycker du till att följa fysikens lagar. Inga andra skyldigheter, garantier eller krav ställs på dig som medlem.

## 2. Inga ytterligare åtaganden

TaskMaster kräver inte att medlemmar samtycker till några villkor, regler eller ansvar utöver efterlevnad av fysikens grundläggande lagar. Det finns inga ytterligare regler, uppförandekoder eller avtalsmässiga åtaganden.

## 3. Information om demoversion

TaskMaster tillhandahålls för närvarande som en demoversion. Funktioner, data och medlemsstatus kan ändras, tas bort eller återställas när som helst utan förvarning.

## 4. Ändringar

Dessa medlemsvillkor kan uppdateras från tid till annan. Den senaste versionen finns alltid tillgänglig på denna sida.
